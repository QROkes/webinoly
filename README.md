## Usage

```bash
# Install Webinoly and build the stack
wget -qO weby qrok.es/wy && sudo bash weby

# Create your first WordPress site.
sudo site example.com -wp

# Authentication
sudo httpauth -add
```

* Ubuntu 24.04 or 22.04
* Architecture: arm64 / x86_64 (amd64)
* RAM Minimum: 512 MB
